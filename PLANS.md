- 为 `血液与肿瘤/PBL作业/Case1-第二幕/Case1-第二幕.md` 生成一套可复用的 Markdown 转 HTML/PDF 脚本。
- 复用并改造第一幕的视觉样式，使第二幕支持新的行内链接格式、块引用样式、阶段总结标题与保留导语段落。
- 在第二幕目录输出 `style.css`、`build_html.py`、美化后的 HTML 以及 A4 PDF。

## 目标拆解
1. 盘点第一幕旧脚本的硬编码逻辑，找出与第二幕结构不兼容的部分。
2. 设计通用解析规则：
   - 直接解析新的 `[显示文字](链接 "悬浮标题")` 行内链接；
   - 保留“先说我觉得最重要的机制问题”这类问题区前导段落；
   - 将 Markdown 块引用 `>` 渲染为黄色提示框；
   - 兼容 `本幕总结（当前阶段结论）` 这类新的总结标题；
   - 保留题目区、关键词区、问答区、总结区的版式。
3. 设计导出流程：
   - HTML 输出默认落在当前 Markdown 同目录；
   - PDF 采用 A4，保留背景颜色与块元素底色；
   - 无法导出 PDF 时给出明确依赖提示，但不影响 HTML 生成。
4. 运行脚本，对 HTML 与 PDF 进行实际检查，修正兼容问题。

## 常见坑 / 复现方法
- 坑1：`git status` 对未跟踪目录只显示目录名，不会直接展开里面的文件。（复现：新建目录后直接运行 `git status --short`。）
- 坑2：Windows 默认 Git 可能转义中文路径，阅读状态时需要关闭 `quotepath`。（复现：直接运行未设置 `core.quotepath=false` 的状态命令。）

## 一、课程导入：这门课为什么重要

老师开场说明：今天开始讲一门新课——**心理危机干预与预防**（**psychological crisis intervention and prevention**）。

老师先用很口语化、很直接的方式指出：
大学生的心理危机在极端情况下，常表现为两类后果：

1. **自杀（suicide）**：伤害自己、结束生命。
2. **伤人/杀人（homicide / violence toward others）**：把危机转向他人。

虽然这类事件总体上并不常见，但一旦发生，社会影响非常恶劣。
因此，这门课不只是完成一门课程，更重要的是：

* 学会**识别（identification）**
* 学会**调整（adjustment）**
* 学会**助人自助（help others and help oneself）**

核心目的还是“健康平安”。

## 三、老师展示个人科普文章：大学生抑郁症要重在早识别、早预防

老师提到左侧一本杂志，是中国心理卫生协会主办的科普杂志《心理与健康》。
其中某年（老师说为**2023年5月**）邀请其撰写封面文章，主题是：

**大学生抑郁症早期识别与预防**（early identification and prevention of depression in college students）

### 老师强调的背景判断

疫情以后，大学生**抑郁症（depression）**已经成为大学生**休学、退学**的重要原因，甚至是主要原因之一。

老师回顾说：

* 过去学生休学退学原因中，较常见的是传染病、外伤等；
* 现在心理问题尤其突出；
* 抑郁症一旦发展起来，**不容易恢复**，因此更应强调**预防（prevention）**。

### 【学习建议】

凡是“不好治”的病，都要强调预防。
对抑郁症而言，关键不是等严重了再处理，而是：

* 稍有苗头就要注意；
* 稍有症状就应及时发现；
* 及时调整，避免进一步加重。

## 四、政策背景：健康第一，尤其要重视“心理脆弱”

老师提到，国家层面已明确强调要重视**心理健康（mental health）**与**精神卫生（mental hygiene / mental health service）**。

并提到教育系统当前强调“**健康第一**”，不是“学习第一”。

### 老师提到的“三小”问题

当前学生群体，尤其是中小学生中，突出有“三小”：

1. **小眼镜**：近视问题突出；
2. **小胖墩儿**：肥胖问题增加；
3. **小脆心儿**：心理脆弱、抗挫能力差。

老师的意思是：
过去很多家长、学生把“学习”放在第一位，甚至牺牲健康去换成绩；现在越来越清楚，**健康没了，学习最后也守不住**。所以必须把“健康第一”摆到前面。

## 五、从“危机（crisis）”讲起：危机的广义概念

老师先从一般性的“危机”讲起。
危机可理解为：某种事件使社会偏离正常运行轨道，对公共安全、稳定造成重大影响的一种**非均衡状态（state of disequilibrium）**。

### 危机按来源大致可分三类

1. **天灾**：如地震、洪水、山火等；
2. **人祸**：如战争；
3. **自然因素与人为因素叠加**：人为因素加重自然危机。

老师这里举了若干国际和自然灾害新闻作为引入，目的不是让大家背新闻本身，而是让大家理解：
**危机状态意味着安全感被打破，日常秩序被中断，人会进入高度不确定与高压力状态。**

### 老师的引申

大家能坐在教室里正常上课、没有寒冷和生命危险，其实已经是一种相对稳定、安全的状态。
而在地球另一些地方，人可能随时面对死亡威胁，这就是危机情境。

## 七、【考点】心理危机（psychological crisis）的定义

老师明确说：**今天第一个重点，就是心理危机的概念。**

### 【考点】心理危机定义

**心理危机（psychological crisis）**：
指个体面临突然的或重大的生活困境，**无法用既有方式解决**，从而出现**心理失衡（psychological disequilibrium）**的状态。

### 【考点】心理危机成立的三条标准

老师说，判断心理危机，要同时抓住三条：

#### 1. 有一个“事儿”——即存在生活事件（life event）

必须先有一个外部或内部触发的重大事件、困境或打击。

#### 2. 出现严重反应

包括多方面的严重心身反应：

* **情绪反应（emotional response）**
* **认知反应（cognitive response）**
* **身体反应（physical / somatic response）**
* **行为反应（behavioral response）**

#### 3. 不能有效自我缓解

也就是：

* 不能靠现有经验、办法、知识把问题解决掉；
* “老办法解决不了新问题”；
* 已超出个体固有能力边界。

### 老师总结式表述

心理危机本质上就是：
**有重大事件 + 有严重反应 + 自己缓不过来。**

## 八、大学生常见心理危机的极端形式：以自杀为主

老师再次回到大学生群体，指出大学生常见心理危机的极端形式之一就是**自杀（suicide）**。

### 老师对本校与兄弟院校的比较性评论

老师说，近几年本校没有自杀成功案例，是比较幸运的。
但其他高校并不都如此，尤其一些学校学生多、竞争强、卷得厉害，相关问题更突出。

### 老师的解释

* 学校健康教育抓得相对较好；
* 医学生多学了一些健康知识；
* 更有利于自我识别和自我保护。

## 九、【举例 / 临床关联】天津师范大学女大学生因乙肝“大三阳”被孤立后自杀案例

老师说，单讲抽象概念大家容易困，所以开始举案例。
他说明：许多真实案例涉及工作机密，不能讲，因此选了一个网上公开案例。

### 案例基本情况

* 学校：天津师范大学
* 对象：一名大一女生
* 转写中名字前后有异写，这里按同一案例整理

### 事件经过

该女生入学后，在一次偶然检查中发现自己是**乙肝大三阳**。
老师指出：她本人可能当时并无明显症状，但被检查出有病毒、且具有传染性后，同学开始疏远、孤立她。

### 【医学相关】

老师这里提到：

* 乙肝大三阳与**乙型肝炎（hepatitis B）**相关；
* 传播主要与**体液传播（body fluid transmission）**有关；
* 并不是像呼吸道传染病那样通过日常呼吸轻易传播。

老师的意思是：
真正传播风险未必像大家想象得那样大，但因为同学不懂医学知识，所以恐惧、疏远仍然发生了。

### 被孤立后的变化

* 吃饭、上课、生活中被刻意拉开距离；
* 逐渐被同学群体排斥；
* 心理压力越来越大。

### 学校与辅导员的处理

辅导员曾劝其先休学治病。
但该女生认为：

1. 自己没症状；
2. 大三阳不是短时间就能治好的；
3. 家里贫困，休学意味着拖延毕业、增加经济负担；
4. 患病不能成为剥夺其受教育权的理由。

### 学校后续安排

在宿舍紧张情况下，学校为“照顾”她，给了一个**单间宿舍**。

老师指出：
表面上看，像是“优待”；但实际上，这种安排也可能进一步加重其**隔离感、孤独感**。

### 结局

她于3月入住单间，一个月后在宿舍内**烧炭自杀**。

### 遗书内容与心理状态

老师转述其遗书意思：

* 对未来彻底绝望；
* 感到人生漫长而看不到尽头；
* 想从当前“腐坏”的生活中尽快逃离；
* 也知道“不应效仿”，但自己已救不了自己。

### 老师对其心理逻辑的概括

她感到：

* 因疾病被孤立；
* 眼前看不到希望；
* 未来读书、就业、社会交往似乎都会继续遭受歧视；
* 于是选择“尽快逃离”。

### 【易错 / 识别点】

老师补充说：

* 她不是没有努力过，也尝试向朋友解释疾病并不可怕；
* 但新生群体对传染病不了解，恐惧和流言仍在扩散；
* 自杀当天其母亲已有不祥预感，曾联系辅导员；
* 但因误判她人在图书馆，错过了抢救关键时机。

### 引发的争议

事件后来引起很大争议，焦点包括：

* 患有传染病的学生应如何安排；
* 学校在保障其权利与保护其他学生之间如何平衡；
* 辅导员、学校是否已尽责；
* 同学的排斥与恐惧在其中起了多大作用。

### 老师的评价

老师认为：

* 这是一个由疾病引发、再被孤立放大的心理危机；
* 其中有老师和学校的责任；
* 但客观说，学校也并非完全不作为，而是提供了某种支持；
* **真正困难的是：心理危机的预防并不容易。**

### 【助人者视角】

老师引用该校心理中心主任在事件后的朋友圈感慨，大意是：
自杀像地震，很多征兆往往只有事后追溯时才显得具有预测意义。
每个生命的逝去，都会让助人者感到悲伤、内疚、悔恨，以及无奈、无助。

## 十、【考点】危机干预（crisis intervention）的概念、目的与意义

讲完案例后，老师转入理论。

### 【考点】危机干预的提出

老师说，**危机干预（crisis intervention）**概念最早由美国心理学家**凯普兰（Caplan）**于**1954年**提出，并进行系统研究。

### 【考点】危机干预的定义

危机干预属于广义的**心理治疗（psychotherapy）**范畴。
其核心是运用心理治疗的手段，帮助处于危机状态的当事人：

* 处理迫在眉睫的问题；
* 恢复心理平衡；
* 安全度过危机；
* 重新适应生活。

### 【考点】危机干预的两个主要目的

1. **避免自伤或伤人**

   * 防止自杀、自残
   * 防止伤及他人

2. **恢复心理平衡与心理动力**

   * 使人重新“运转起来”
   * 不再陷于崩溃状态

### 【考点】危机干预成功的意义

如果危机干预成功，个体可以：

1. 更好地把握自己的现状；
2. 重新认识危机事件；
3. 学会未来再遇到类似危机时更好的应对策略。

## 十一、世界范围内的自杀情况：老师用于引入的流行病学比较

老师接着从全球范围谈自杀问题，作为宏观背景。

### 老师课堂中提到的比较

* 自杀率较高的国家：韩国、匈牙利、日本等
* 自杀率较低的国家：冰岛、西班牙、希腊等
* 自杀是一个**概率事件**，没有哪个国家是绝对零发生

### 老师进一步讲的趋势

从纵向看，西方国家自杀率在历史上有：

* 两次低谷：两次世界大战时期
* 两次高峰：经济大萧条相关时期

老师借此强调一个朴素判断：
**经济压力、贫困、失业与心理危机密切相关。**

### 老师的口语化总结

“钱不是万能的，但没有钱是万万不能的。”
他用这个意思强调：
收入和现实生活条件，确实能缓解大量现实压力和心理问题。

## 十二、中国大学生自杀率：老师强调“总体不高，但趋势需警惕”

老师提到：

* 美国总体自杀率居中；
* 美国大学生自杀率大致低于其全人群平均；
* 中国总体自杀率也有自己的水平；
* 中国大学生自杀率相对更低。

### 老师的解释

中国高校自杀率相对较低，老师将其归因为：

* 学校对心理健康工作的高度重视；
* 有专门心理老师；
* 有辅导员体系；
* 对学生有更强的组织性和日常管理。

### 【复习 / 观点】

老师顺带比较了中国大学与国外大学的学生管理差异：

* 国外大学多是个人化、分散式管理；
* 没有中国式的班级、班委、辅导员等强集体支持系统；
* 中国高校的班集体、宿舍集体，在心理危机预防上反而是一种保障。

老师要表达的是：
**一个人出问题时，若身处集体，更容易被发现、被支持。**

## 十三、天津市学生自杀的趋势：疫情后出现低龄化

老师提到，天津市每年具体自杀人数属于保密信息，但他给出了几个趋势判断：

### 趋势

1. **总体逐年增长**
2. **疫情后结构变化明显**

   * 疫情前：非正常死亡学生中，研究生和本科生占比更高
   * 疫情后：中小学生占比明显上升
3. **低龄化**

   * 老师提到最小案例已到小学四年级

### 老师结论

当前形势不容乐观。

## 十四、北京某调查总结的大学生自杀规律

老师说，最新数据未必掌握，但举了一个较早的北京研究作参考。
该研究收集了**200例大学生自杀相关个案**，从中总结了一些规律。

### 1）自杀地点分布

老师提到，地点以以下场所为主：

* **宿舍**最多
* 其次是**教学楼**
* 还有**科研楼、家属楼**等

### 解释

说明自杀行为常发生在学生**最熟悉、最常活动**的场所。

### 2）跳楼楼层分布

老师说，自杀跳楼楼层较集中在：

* 5楼
* 6楼
* 7楼
* 8楼

其中**7楼最多**。

老师用建筑标准与致死概率作口语化解释：

* 太低可能摔不死；
* 七层楼又常是学校建筑的典型高度。

### 3）最主要原因：不适应（maladjustment）

老师强调，“不适应”是非常核心的原因，而且是**各种不适应叠加**：

* 学业不适应
* 人际不适应
* 气候不适应
* 饮食不适应
* 环境不适应
* 专业不适应

#### 【举例】

* 从小学到初中、高中到大学，角色变化导致失落；
* 原来在原学校很优秀，换环境后优势消失；
* 第一本志愿没录上，被调剂到不喜欢的专业；
* 想退学重考，又没勇气，长期纠结。

### 4）抑郁（depression）

老师说，很多学生已经处于抑郁状态，但：

* 自己不知道；
* 或者不承认；
* 硬撑、逞强，最后越来越严重。

### 5）危机发生时间

老师强调一个容易与直觉不符的点：

* 大家往往以为夜里更容易出事；
* 但调查提示：**白天比晚上多**
* 尤其是**上午**更容易发生。

### 6）自杀方式

最显著高于其他方式的是：**跳楼**

老师给的解释是：

* 不需要额外工具；
* 致死率高；
* 操作相对“简单”。

### 7）年龄与年级分布

老师指出：

* 中小学阶段：高年级更突出
* 大学阶段：**随着年级升高，危机数量逐年增加**
* **毕业年级最高发**

### 老师对毕业年级高发的解释

毕业班问题交织最复杂：

* 能否毕业
* 找工作困难
* 考研压力
* 答辩、盲审压力
* 恋爱与分手问题
* 多重现实压力集中爆发

### 8）性别差异

老师说，北京这项研究提示：

* 男女比例基本接近 **1:1**
* 自杀方式差异也不大
* 只是时间选择上略有不同：

  * 男生夜间稍多
  * 女生白天稍多

---

## 十五、费立鹏关于中国人自杀原因的总结

老师接着提到加拿大心理学家**费立鹏（Michael R. Phillips）**在中国开展研究，并总结出中国人自杀的八大相关因素。

### 八大原因

1. **抑郁程度重**
2. **有自杀未遂史**
3. **应激强度大（high stress intensity）**
4. **生命质量低**
5. **慢性心理压力大**
6. **死亡前两天内有人际冲突**
7. **有血缘关系的人存在自杀行为**
8. **熟人有自杀行为**

### 【考点理解】

老师特别强调第6条很有“中国特色”：
中国人的自杀往往与**亲密关系中的激烈冲突**有关，很多是“吵起来了”“怼起来了”之后的**激情性自杀**。

---

## 十六、中国人与国外自杀模式的差异

老师根据费立鹏研究，讲了几组对比。

### 1）性别差异不同

老师的课堂表述是：

* **中国**：女性自杀多于男性
* **国外**：男性自杀多于女性

老师解释认为：
中国总体数据受**农村女性自杀率较高**影响很大。

### 2）老师对农村女性高自杀率的解释

他认为与以下因素有关：

* 女性社会地位相对较低；
* 封建观念残余；
* 家庭矛盾无法有效解决；
* 遇到不公和挫折时缺乏支持与出路。

### 【举例】

老师以**彩礼**为例，从其角度批评这是一种对女性的物化和歧视。
并指出在某些家庭情境中，女性在婚后面临：

* 婆媳矛盾
* 夫妻冲突
* 娘家、婆家双重压力
* 离也离不了，过也过不下去

最后可能走向极端。

### 3）年龄差异不同

老师说：

* 国外自杀年龄偏大；
* 中国青少年、自青年龄段更突出；
* 中国15～24岁是高风险年龄段之一；
* 平均自杀年龄偏年轻。

### 老师的解释

年轻人之所以更容易自杀，是因为：

* 年龄阶段本身更敏感；
* 长期读书，理想主义色彩强；
* 接触现实后，发现现实与想象不一致；
* 本该“调整主观去适应客观”，但有些年轻人做不到，容易走向极端。

---

## 十七、精神障碍与躯体疾病也是自杀的重要原因

老师接着补充，自杀与**精神障碍（mental disorders）**、**躯体疾病（physical illness / somatic disease）**密切相关。

### 1）抑郁症（depression）

老师强调，抑郁症不是简单的“想不开”，而是一种深度的**精神耗竭（mental exhaustion）**状态。

典型体验包括：

* 持续痛苦
* 大脑像被掏空
* 转不动
* 没动力（loss of motivation）
* 没兴趣（anhedonia）
* 认知越来越消极
* 觉得活着只有痛苦，没有出头之日

对正常人来说，“死”是可怕的；
但对重度抑郁患者来说，死亡可能被体验为“唯一解脱”。

### 2）精神分裂症（schizophrenia）

老师说，**精神分裂症（schizophrenia）**患者也可能自杀，因为会受：

* **幻觉（hallucination）**
* **妄想（delusion）**

支配。
老师还引用电影《追捕》里的情节，帮助理解“被诱导走向死亡”的体验。

### 3）严重躯体疾病

老师提到，一些严重躯体疾病也会增加自杀风险，例如：

* 癌症（cancer）
* 艾滋病（AIDS / HIV infection）等

其相关原因包括：

* 疾病痛苦大；
* 治疗困难；
* 经济负担重；
* 心态崩溃；
* 常伴发抑郁。

---

## 十八、与危机事件相关的常见诱因

老师归纳说，与心理危机相关联的事件，常有以下几类特点。

### 1）持续压力（chronic stress）

例如：

* 家庭经济困难
* 学业困难
* 单亲家庭压力

### 【举例】单亲家庭

老师特别讲到单亲家庭，尤其单身母亲抚养孩子时，容易出现：

* 家长把全部希望投射到孩子身上；
* 对孩子要求过高；
* 孩子一开始努力讨好；
* 之后逐渐达不到期待；
* 最终矛盾升级。

甚至现实中会出现极端的**弑亲**新闻。

### 2）财产、健康、名誉、尊严丧失

老师举到：

* 网络诈骗（telecom / online fraud）
* 健康受损
* 名誉受损
* 尊严被羞辱

### 【举例】网络诈骗

老师提醒：网络诈骗的两个高发受害群体是：

* 老年人
* 大学生

大学生容易成为目标，是因为：

* 社会经验不足；
* 容易轻信；
* 容易被“兼职”“退款”“小便宜”诱骗。

### 【学习建议】

一定要提高警惕。
有的人损失一点钱无所谓，但对另一些人来说，经济打击可能直接把人推入生存危机。

### 3）感情受挫

如：

* 失恋
* 情感挫败
* 情感冲突

### 4）环境改变（environmental change）

如：

* 入学
* 转学
* 搬家
* 留学

这些都可能成为危机高发节点。

### 5）持续的人际关系紧张

包括：

* 夫妻关系
* 家庭矛盾
* 同学关系
* 宿舍关系

### 老师的总结

中国人的自杀，常有**明显的人际关系因素**参与。
这是与一些西方社会相对不同的地方。

---

## 十九、人格（personality）与心理危机的关系

老师进一步说，心理危机不只和外部事件有关，也和人格基础有关。

### 人格的概念

**人格（personality）**是一个人心理特征的总和，像一个相对稳定的“脸谱”。

人格是双刃剑：

* 有优势的一面；
* 也有脆弱、容易出问题的一面。

### 容易出现心理危机者常见人格特点

1. 注意力明显缺乏
2. 过分内省（excessive introspection）
3. 情绪稳定性差
4. 解决问题能力差

### 另一项研究归纳的心理特点

容易出现危机的人，往往还有以下特点：

* 不能客观评价自己，也不能客观评价外界；
* 容易夸大外部危险；
* 对自己评价要么过于自大，要么过度自卑；
* 存在多种**非理性认知（irrational cognition）**：

  * 过度概括（overgeneralization）
  * 非此即彼（all-or-none thinking）
  * 以偏概全
  * 负性推导
  * 过度绝对化、肯定化
* 人际关系不良；
* 人格不成熟；
* 适应能力差；
* 不会求助；
* 不会应对困难局面。

---

## 二十、【举例】两个典型案例：一个杀人，一个自杀

老师说，前面理论有点抽象，接下来用两个典型案例串起来理解：

1. **马加爵**：杀人案例
2. **杨元元**：自杀案例

---

## 二十一、【案例一】马加爵：从优秀贫困生到宿舍杀人案

### 基本情况

* 籍贯：广西
* 学校：云南大学
* 事件：大四最后一个学期前后，杀害4名同学，后逃到三亚，被捕并判死刑

### 老师先强调的一点

马加爵并不是人们想象中的“青面獠牙”式坏人，从外表看反而非常朴实。

### 入学前后的优秀经历

* 从小学到高中成绩都非常优秀；
* 初中参加全国数学竞赛获二等奖；
* 高中考入云南大学，成为村镇骄傲；
* 大一入学后积极报名当班委；
* 热心班级事务，参加活动，承担任务，是积极分子。

---

## 二十二、【案例二】杨元元：从优秀学生干部到研究生宿舍自杀

### 基本情况

* 2009级，上海海事大学法学系硕士生
* 研一入学后不久，于宿舍卫生间自杀

### 自杀方式

老师提到她是用两条毛巾固定后，坐地将自己勒死。
老师用这个细节强调：这说明其求死意志已经非常坚决，因为过程中理论上存在中断机会，但她没有停止。

### 入学前的优秀经历

* 从小学习刻苦，成绩优秀；
* 从高中到大学一直是学生干部；
* 大学期间加入中国共产党；
* 还能帮助同学调解矛盾，是标准的“优秀学生干部”形象。

---

## 二十三、两案共同基础之一：贫困（poverty）

老师说，自己的一家之言是：
这两个人有一个共同基础——**穷**。

### 马加爵的贫困表现

* 吃饭时常只买素菜；
* 有时只靠馒头和免费汤；
* 家里寄钱晚了就借钱吃饭；
* 借不到钱就躺着不动，甚至以睡觉来减少消耗；
* 又穷又好面子，不愿申请贫困资助。

### 杨元元的贫困表现

* 六岁丧父；
* 母亲无固定工作，还要照顾弟弟；
* 她一边上学，一边承担家庭责任；
* 本科毕业时因欠学校贷款，毕业证一度被扣；
* 因没钱甚至卖掉手机，与同学联系中断。

---

## 二十四、两案共同基础之二：学业受挫

### 马加爵的学业受挫

* 大一时非常努力，想拿奖学金；
* 但英语基础差，成为短板；
* 尽管专业课不错，却因英语拖后腿，多次与奖学金失之交臂；
* 原本想通过奖学金同时解决经济和尊严问题，结果两头落空；
* 之后开始躺平、松散。

### 杨元元的学业受挫

* 本科业务成绩第一；
* 但综合测评中其他加分因素影响，最终没能保研；
* 她对这种结果感到委屈、不公平；
* 毕业后又花两年准备考研，但一直没考上；
* 最终是在反复受挫中继续往前顶。

---

## 二十五、两案共同基础之三：感情受挫

### 马加爵

* 喜欢班里一名女同学；
* 在男同学鼓动下尝试表白；
* 打电话被拒；
* 写情书又被当面撕毁；
* 受到强烈打击和羞辱感。

### 杨元元

* 大学时有男朋友；
* 因毕业、考研、压力等因素，关系受挫，最终分手。

---

## 二十六、两案爆发前的关键人际冲突

### 马加爵的爆发点

老师讲到两个关键事件：

#### 1. 被最看重的朋友排除在外

他认为某位老乡兼贫困生同学是自己最好的朋友。
但这位同学过生日时请了几乎所有人，唯独没请马加爵。
这件事对他伤害极大。

#### 2. 打牌时被指“作弊”，并被拿“人品差”羞辱

另一名同学打牌输了后，不仅说他作弊，还拿前面“生日不请他”这件事嘲讽他，说这是因为他“人品不行”。

老师认为，这一下“捅到了肺管子”，彻底触发了他的杀意。

### 之后的发展

* 他买了锤子准备报复；
* 原本计划针对两人；
* 后来另外两名同学“误入现场”，被牵连杀害；
* 还有一名曾在他困难时请过他吃饭的同学，被他放过。

### 【举例中的老师提示】

老师借此强调一句很口语的话：
“勿以善小而不为。”
有时候一个很小的善意，真的可能在关键时刻改变走向。

---

## 二十七、杨元元案中的关键人际冲突：贫困、母亲、宿舍与羞辱感

### 事件背景

杨元元上大学时曾把母亲带到学校附近生活，学校后勤还曾给母亲安排临时保洁工作。
到了上海读研后，她也想继续这样做，让母亲和自己一起生活。

### 冲突升级

但研究生宿舍管理严格，学校不同意。
母亲曾趁宿管不注意进入宿舍，与她同住。
其他同学反映后，学院领导与宿管介入，要求母亲离开。

### 她的主观体验

* 她希望学校能同情她、体谅她；
* 但学校采取公事公办态度；
* 宿管老师多次驱离其母亲；
* 她感到母亲脸面受辱，也感到自己无能；
* 与宿管、学校之间的矛盾不断加深。

---

## 二十八、老师对两案的心理学分析（一）：欲求的绝对性

老师认为，两人一步步走上危机道路，首先与**欲求的绝对性**有关。

### 欲求的绝对性是什么意思

就是对自己、对他人、对事情抱有过于理想化、绝对化的要求。
典型想法包括：

* 人都应该讲理；
* 事情都应该公平；
* 别人应该按我的期待回应我。

### 在马加爵身上的表现

* 我打牌厉害，别人就应该承认并尊重我；
* 我把你当朋友，你就必须也把我当朋友；
* 如果你没有这样做，就是严重背叛与侮辱。

### 在杨元元身上的表现

* 坚信“知识改变命运”；
* 认为只要努力学习，就应该得到改变命运的结果；
* 明明资源不够、条件不够，却执着于一定要考北上广的研究生；
* 认为学校理应同情、帮助自己和母亲；
* 一旦现实没有按理想运行，就难以接受。

### 老师结论

表面看像是“别人逼她/逼他”，但深层常常也是**自己用理想化要求把自己逼到了墙角**。

---

## 二十九、老师对两案的心理学分析（二）：感知的歪曲性

第二个特点是**感知的歪曲性**，即因为贫困、自卑、敏感等基础，在解释外界行为时更容易感到：

* 屈辱
* 被歧视
* 被贬低
* 被看不起

### 马加爵身上的例子

老师举了若干例子：

* 请老乡吃饭时，饭馆老板更重视老乡，不把他当主客；
* 去商场买鞋，被店员默认只能买最便宜的；
* 去银行等候，被保安当成蹭空调的人往外赶；
* 当班委时因常去找团支书汇报工作，被男同学嘲笑想“癞蛤蟆吃天鹅肉”。

老师的意思是：
这些事在一般人看来未必都是“致命打击”，但对一个本就高敏感、高自尊的人来说，会不断叠加成屈辱记忆。

---

## 三十、老师对两案的心理学分析（三）：思维的消极性

老师说，他们在受挫后，思维会迅速走向消极和极端。

### 表现

* 对自己、对他人、对事情都越来越悲观；
* 一旦一件事没达到预期，就容易全面否定；
* 从“很理想主义”一下变成“彻底否定现实”。

### 例子

* 马加爵奖学金拿不到后，开始躺平、逃课、上网；
* 杨元元长期坚持读书考研，发现并未改变命运后，开始彻底怀疑自己全部努力的意义。

---

## 三十一、老师对两案的心理学分析（四）：应对的非理性

第四个问题是**应对非理性（irrational coping）**。
即在面对压力和挫折时，不会调整路线，不会换方法，不会求助，而是用极端方式解决问题。

### 马加爵

被捕后自述大意是：

* 贫困和艰辛我能忍；
* 别人的歧视、蔑视我也能忍；
* 但最好的朋友践踏我的人格尊严，我不能忍；
* 既然这样，就“玉石俱焚”，要给那些歧视贫穷、歧视苦人的人一个教训。

老师强调：
其核心是**自尊受伤**后，以极端暴力方式“讨回尊严”。

### 杨元元

老师转述其遗书中的一句核心态度：
**“人可以被毁灭，但不可以被打败。”**

老师借此指出：
这类人往往把“宁折不弯”当成尊严，但如果不会调整和转弯，就可能把自己逼向绝路。

---

## 三十二、老师总结两案共有的人格与心理特点

老师把两人的共性概括为三个词：

### 1. 高敏感

* 对人、对事太在乎；
* 容易把小事放大；
* 对羞辱、轻视体验格外强烈。

### 2. 高自尊

* 内里可能自卑；
* 但越自卑，越想通过成绩、表现、荣誉来证明自己；
* 追求高自尊的过程，反而不断受伤。

### 3. 执着、倔强

* 一条路走不通，也不愿转弯；
* 资源不够，却设定很高目标；
* 遇到困难不知调整；
* 坚持到最后，把自己“绷断”。

### 老师的概括性结论

本质上是：

* **资源不足**
* 却定了过于理想的目标
* 遇到挫折不懂调整
* 最后把自己逼崩了

老师认为，这种教训是很惨痛的。

---

## 三十三、【举例 / 本校案例】过度要强、宿舍关系差、学习方法僵化，最终精神失常边缘

老师接着举了一个本校女同学的危险案例，用来提醒大家重视现实中的前兆。

### 基本情况

* 女同学
* 本硕连读
* 高中时是学霸
* 家长在原学校还是主任级人物
* 从前荣誉很多，长期处于“优秀学生”位置

### 入学后的目标

她进入医大后，给自己偷偷立了一个很高的标杆：

* 要拿奖学金

function ensureArray(obj, key) {
  if (!Array.isArray(obj[key])) {
    obj[key] = [];
  }
}

/* -------------------------
 * 工具函数：按 name 去重插入
 * 如果已存在同名项，就替换；否则追加
 * 适用于 proxies / proxy-groups
 * ------------------------- */
function upsertByName(arr, item) {
  const index = arr.findIndex(x => x && x.name === item.name);
  if (index >= 0) {
    arr[index] = item;
  } else {
    arr.push(item);
  }
}

/* -------------------------
 * 工具函数：前置插入规则并去重
 * ------------------------- */
function prependUniqueRules(oldRules, newRules) {
  const oldList = Array.isArray(oldRules) ? oldRules : [];
  const oldSet = new Set(oldList);

  // 只保留那些旧规则里还没有的新增规则
  const uniqueNewRules = newRules.filter(rule => !oldSet.has(rule));

  return uniqueNewRules.concat(oldList);
}
